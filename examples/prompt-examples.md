# Prompt Examples

## 1. Project Blog Draft

```text
Use $csdn-tech-blog-writer to analyze this Spring Boot project and generate a full CSDN article.
Focus on project structure, core business flow, database design, and API layering.
```

## 2. Knowledge-Point Blog

```text
Use $csdn-tech-blog-writer to explain this ThreadLocal helper class as a CSDN knowledge-point article.
Please include principle analysis, usage examples, pitfalls, and best practices.
```

## 3. Outline First

```text
Use $csdn-tech-blog-writer to read this controller, service, mapper, and SQL, then give me only an outline.
I want a project-practice article outline, not the full draft yet.
```

## 4. Polish Existing Draft

```text
Use $csdn-tech-blog-writer to polish the following article draft.
Keep my original tone, but improve clarity, section flow, and code explanation quality.
```

## 5. Style-Adaptive Writing

```text
Use $csdn-tech-blog-writer to analyze these two past blog posts and then write a new CSDN article in the same style.
Do not copy old sentences directly. Keep the rhythm, structure, and tone only.
```

## 6. Multi-Post Series

```text
Use $csdn-tech-blog-writer to split this project into a 3-part CSDN series.
Part 1 should cover architecture, part 2 should cover core CRUD logic, and part 3 should cover optimization and pitfalls.
```
